Varför

Vår webbsida erbjuder möjligheten att kunna se och jämföra B12 i livsmedelsprodukter. Du som användare kan skapa din egen profil och redan idag öka medvetenheten kring vad du äter. Det är inte alltid så att kroppen säger ifrån i tid om visst näringsämne saknas och därför erbjuder vi en webbtjänst där du lätt kan överblicka över vad du får i dig och lika viktigt, vad du inte får i dig.

Hur

Vår webbtjänst erbjuder möjligheten att kunna se näringsvärdent i en eller flertal matvaror. Detta är möjligt genom att vi använder oss av ett API från Livsmedelsverket som ger tillgång till deras databas på över 2000 olika livsmedel med 50 olika typer av näringsvärden. Livsmedel i databasen är klassificerade enligt den europeiska standarden LanguaL och varje livsmedel ingår i en klassificering och har ett unikt id.

Om

Hemsidan är byggd på HTML (UTF-8) och CSS för att visuellt kunna anpassa hemsidan efter vårt behov, vidare används JavaScript med jQuery-bibliotek som tillhandahåller ramverket Ajax. Detta för att kunna utöka användbarheten för både utvecklare samt slutanvändaren genom att implementera exempelvis knappar med tydlig ”feedback” till användaren och andra tekniska och grafiska signaler för att öka sidans funktionalitet.
